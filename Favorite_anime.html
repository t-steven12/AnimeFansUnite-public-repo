ADD NEW/DISPLAY Users' Favorite Animes

The Users' Favorited Animes page regulates the Fav_animes table.

This page displays a table of all animes favorited by a user.
The Fav_animes table bridges the M:M relationship between Users
and Animes. A user can view a user's favorite animes by searching their
user ID. "User's Favorite Animes" is to be populated by the search results.
An admin/User can also favorite an anime by submitting a User ID and
Anime ID in the section below. This inserts a row into the Fav_animes table.
Deleting a User will remove all of the deleted User's associated rows in
the Fav_animes table. The associated Animes of the removed rows will be unaffected.
